## Getting Started

### Creating Your First Post

1. Click **New Post** from the dashboard
2. Enter your title
3. Start writing in the rich text editor
4. Save as draft or publish immediately

### Editor Features

- **Rich Text Formatting**: Bold, italic, headers, lists
- **Media Embeds**: Images, videos, tweets, code blocks
- **Auto-save**: Drafts saved every 30 seconds
- **Version History**: Restore previous versions

## Writing Posts

### Using the Rich Text Editor

**Formatting**:
- `/bold` or `Ctrl+B` - Bold text
- `/italic` or `Ctrl+I` - Italic text
- `/code` -Inline code
- `/heading1` - Large heading
- `/heading2` - Medium heading
- `/heading3` - Small heading

**Lists**:
- `/bullet` - Bullet list
- `/number` - Numbered list
- `/check` - Checklist

**Media**:
- `/image` - Insert image
- `/video` - Embed video
- `/code-block` - Code snippet
- `/quote` - Block quote

### Embedding Content

**Images**:
1. Type `/image` or click image icon
2. Upload file or paste URL
3. Add alt text for accessibility
4. Adjust size and alignment

**Videos**:
- YouTube: Paste URL, auto-embeds
- Vimeo: Paste URL, auto-embeds
- MP4: Upload file directly

**Code Blocks**:
```
Type /code-block
Select language (JavaScript, Python, etc.)
Paste your code
```

**Social Media**:
- Twitter: Paste tweet URL
- Instagram: Paste post URL
- GitHub Gists: Paste gist URL

### Draft Auto-Save

- Saves every 30 seconds automatically
- Manual save: `Ctrl+S`
- View all drafts: **Dashboard** → **Drafts**
- Recover deleted draft: **Trash** (30-day retention)

### Version History

Access previous versions:
1. Click **History** button
2. Browse all saved versions
3. Preview changes
4. **Restore** to any version

## SEO Optimization

### Meta Description

**Auto-Generated**:
- First 160 characters used automatically
- Customize in **SEO Settings** panel

**Best Practices**:
- Keep under 160 characters
- Include primary keyword
- Make it compelling

**Customize Cards**:
1. Open **Social Preview** tab
2. Upload custom image (1200x630px recommended)
3. Edit title and description
4. Preview for Twitter, Facebook, LinkedIn

- Auto-generated from title

## Troubleshooting

### Auto-Save Not Working

**Solutions**:
1. Check internet connection
2. Clear browser cache
3. Disable browser extensions
4. Try incognito mode

### Images Not Uploading

**Check**:
- File size (max 10MB)
- File format (JPG, PNG, GIF, WebP)
- Network connection
- Storage quota

### Version History Missing

- Only available for published posts
- Restoring requires editor role
- History kept for 90 days

### SEO Preview Not Accurate

- Clear cache and regenerate
- Check meta description length
- Verify image dimensions
- Test with Google's Rich Results Test

### Embed Not Loading

**Verify**:
- URL is public
- Platform is supported
- No ad blockers interfering
- Try different embed method
