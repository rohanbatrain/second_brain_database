## Getting Started

### Creating a Family

1. Go to **Family Hub** → **Create Family**
2. Enter family name
3. Add family members by email
4. Set up initial roles

### Inviting Members

**Send Invitations**:
1. **Family Settings** → **Members**
2. Click **Invite Member**
3. Enter email address
4. Select role (Parent/Child)
5. Send invitation

**Roles**:
- **Parent** - Full access, can manage all features
- **Child** - Limited access, can view and complete tasks

## Calendar Management

### Creating Events

1. Click **+** on calendar
2. Fill in event details:
   - Title
   - Date and time
   - Event type (Family/Personal/Task)
   - Location (optional)
   - Description
3. Click **Create Event**

### Event Types

**Color-Coded**:
- **Family** (Blue) - Family gatherings, dinners
- **Task** (Amber) - Task deadlines
- **Personal** (Purple) - Individual appointments

### View Modes

**Switch Between**:
- **Month** - Full month overview
- **Week** - Detailed weekly view
- **Day** - Hourly schedule
- **Agenda** - List of upcoming events

**Your Choice is Saved**: The calendar remembers your preferred view.

### Drag-and-Drop Rese scheduling

1. Click and drag an event
2. Drop it on new date/time
3. Conflict detection activates
4. Confirm if conflicts exist

### Conflict Detection

**Automatic Warnings**:
- Detects overlapping events
- Shows conflicting events
- Option to proceed or reschedule

**Example**: Trying to schedule "Soccer Practice" when "Piano Lesson" is already  booked will show a warning.

### Recurring Events

**Create Repeating Events**:
1. Create event normally
2. Enable **Recurring**
3. Choose pattern:
   - Daily
   - Weekly (select days)
   - Monthly
   - Custom
4. Set end date

### Event Reminders

**Automatic Notifications**:
- 1 week before
- 1 day before
- 1 hour before

**Customize**: Edit event → Reminders tab

### Calendar Export

**Export to iCal**:
1. Click **Export** button
2. Select date range
3. Download `.ics` file
4. Import to Google Calendar, Apple Calendar, etc.

## Task Management

### Creating Tasks

1. Click **Create Task**
2. Enter details:
   - **Title** - What needs to be done
   - **Description** - Additional details
   - **Assigned To** - Family member
   - **Priority** - Low/Medium/High
   - **Reward** - SBD tokens
   - **Due Date** - Optional deadline
   - **Categories** - Organize tasks
3. Click **Create**

### Task Priorities

**Visual Indicators**:
- **High** - Red badge
- **Medium** - Yellow badge
- **Low** - Blue badge

### Categories

**Common Categories**:
- Chores
- Homework
- Shopping
- Errands
- Education

**Custom Categories**: Add your own

### Subtasks

**Break Down Large Tasks**:
1. Open task details
2. Click **Add Subtask**
3. List smaller steps
4. Track completion:50%

**Example**:
- Task: "Clean Garage"
  - Subtask: "Organize tools"
  - Subtask: "Sweep floor"
  - Subtask: "Take out trash"

### Task Dependencies

**Set Prerequisites**:
1. Create task
2. Select **Depends On**
3. Choose prerequisite tasks
4. Dependent task can't complete until prerequisites done

**Example**: "Finish homework" depends on "Buy textbooks"

### List vs. Kanban View

**List View**:
- Pending Tasks
- In Progress
- Completed

**Kanban Board**:
- Visual columns
- Drag tasks between statuses
- Great for visual learners

**Toggle**: Click view buttons at top

### Filtering Tasks

**Filter By**:
- **Priority** - Show only high priority
- **Category** - View specific categories
- **Assignee** - See your tasks

**Multiple Filters**: Combine for precise views

### Completing Tasks

1. Find pending task
2. Click **Mark Complete**
3. Earn SBD tokens (if reward set)
4. Task moves to completed section

## Token System

### Earning Tokens

**Complete Tasks**:
- Each task has reward amount
- Tokens added to balance immediately
- View balance in header

**Token Rewards Example**:
- Clean room: 10 SBD
- Do homework: 15 SBD
- Help with dinner: 5 SBD

### Spending Tokens

**Token Marketplace**:
1. Browse available rewards
2. Check your balance
3. **Purchase** with tokens
4. Redeem rewards

**Sample Rewards**:
- Extra screen time (50 SBD)
- Choose dinner (100 SBD)
- Movie night (150 SBD)
- Later bedtime (75 SBD)

### Token Transfers

**Send Tokens to Family Members**:
1. Click **Transfer**
2. Select recipient
3. Enter amount
4. Add message (optional)
5. Confirm transfer

**Use Cases**:
- Gift tokens
- Allowance distribution
- Rewards for non-task achievements

### Transaction History

**View All Activity**:
- **Tokens** → **History**
- See all earned/spent tokens
- Filter by date
- Export to CSV

### Allowance Automation

**Set Up Regular Allowance**:
1. **Family Settings** → **Allowance**
2. Set amount per member
3. Choose frequency (weekly/monthly)
4. Tokens auto-distributed

## Troubleshooting

### Calendar Events Not Syncing

**Solutions**:
1. Refresh page
2. Check family membership
3. Verify event permissions
4. Clear browser cache

### Tasks Not Showing

**Check**:
- Selected filters (remove all filters)
- Family selection
- Date range
- Deleted tasks in trash

### Tokens Not Updating

**Verify**:
- Task completion confirmed
- Reward amount set
- Network connection
- Refresh balance

### Cannot Invite Member

**Common Issues**:
- Email already in use
- Reached member limit
- Invalid email format
- Must be parent role to invite

### Export Not Working

- Check date range selected
- Ensure events exist in range
- Try different browser
- Disable pop-up blocker

## Best Practices

###  Task Management

- Set realistic deadlines
- Break large tasks into subtasks
- Use priorities appropriately
- Review completed tasks weekly

### Token Economy

- Fair rewards for effort
- Consistent token values
- Don't overinflate rewards
- Discuss token usage together

### Calendar Organization

- Color-code consistently
- Set up recurring events for routines
- Use descriptive titles
- Review calendar weekly as family

### Family Communication

- Regular family meetings
- Celebrate completions
- Encourage task ownership
- Positive reinforcement
