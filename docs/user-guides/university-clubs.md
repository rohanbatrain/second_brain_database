## Getting Started

The University Clubs Platform allows students and faculty to create, join, and manage university clubs.

### Prerequisites

- Active university account
- Verified email address
- Student/faculty status

### First Steps

1. **Complete your profile** - Add bio, interests, and contact info
2. **Browse clubs** - Explore existing clubs
3. **Join clubs** - Request membership
4. **Create a club** - Start your own club (requires approval)

## Creating and Managing Clubs

### Creating a Club

1. Navigate to **Clubs** → **Create New Club**
2. Fill in club details:
   - **Name** - Club name (must be unique)
   - **Description** - Brief overview
   - **Category** - Academic, Sports, Arts, Social, etc.
   - **Privacy** - Public or Private
3. Upload club logo/banner
4. Set membership requirements
5. Submit for review

**Approval Process**: Admin reviews within 2-3 business days

### Club Roles

- **President** - Full control, can assign roles
- **Vice President** - Assists president, manages operations  
- **Treasurer** - Manages budget and finances
- **Secretary** - Handles communications and records
- **Member** - Basic participation rights

### Editing Club Information

Presidents and VPs can edit:
1. Go to **Club Dashboard** → **Settings**
2. Update description, logo, or privacy settings
3. Click **Save Changes**

## Member Management

### Managing Member Roles

**Presidents can**:
1. Go to **Members** tab
2. Click on member name
3. Select **Change Role**
4. Choose new role from dropdown
5. Confirm assignment

### Activity Tracking

Track member engagement:
- **Event Attendance** - View attendance history
- **Contribution Score** - Based on participation
- **Join Date** - When they joined
- **Last Active** - Most recent activity

### Badges and Achievements

Reward active members with badges:
- **Founding Member** - Original club members
- **Event Champion** - Attended 10+ events
- **Active Contributor** - Regular participation
- **Leadership** - Served in officer role

**Assigning Badges**:
1. **Members** → **Select Member** → **Awards**
2. Choose badge
3. Add optional note
4. Assign

### Onboarding New Members

1. **Welcome Message** - Auto-sent upon joining
2. **Orientation Tasks** - Checklist for new members
3. **Buddy System** - Assign mentor (optional)
4. **First Week Activities** - Suggested events

## Events

### Creating Events

1. **Events** → **Create Event**
2. Fill in details:
   - **Title** - Event name
   - **Description** - What to expect
   - **Date & Time** - When it happens
   - **Location** - Where (virtual or physical)
   - **Capacity** - Max attendees (optional)
3. Add event image
4. Publish or save as draft

### Event Calendar

**Views**:
- **Month View** - Overview of all events
- **Week View** - Detailed weekly schedule
- **Day View** - Hourly breakdown
- **List View** - Chronological list

**Features**:
- **Color Coding** - By event type
- **Drag-and-Drop** - Reschedule events
- **Quick Add** - Click any date to create

### Recurring Events

Create repeating events:
1. When creating event, enable **Recurring**
2. Choose pattern:
   - Daily
   - Weekly (select days)
   - Monthly (by date or day)
   - Custom pattern
3. Set end date or number of occurrences

### Event Reminders

Members receive reminders:
- **1 week before** - Save the date
- **1 day before** - Tomorrow reminder  
- **1 hour before** - About to start

**Customizing Reminders** (Presidents):
- **Event Settings** → **Reminders**
- Toggle reminder types
- Set custom timing

### Event Templates

Save time with templates:
1. **Events** → **Templates**
2. **Create Template** or modify existing
3. Fill in reusable details
4. Save template

**Using Templates**:
- **Create Event** → **Use Template**
- Select template
- Modify as needed

### Photo Galleries

Share event memories:
1. After event, go to **Event Page**
2. **Upload Photos**
3. Select multiple images
4. Add captions (optional)
5. Publish gallery

Members can browse and download photos.

## Communication

### In-App Messaging

**Direct Messages**:
1. Click member avatar
2. **Send Message**
3. Type message
4. Send

**Group Chats**:
- Auto-created for each club
- Officers have separate channel
- Event-specific chats

### Announcements

Broadcast to all members:
1. **Club Dashboard** → **Announcements**
2. **New Announcement**
3. Write message
4. Choose recipients:
   - All members
   - Specific roles
   - Event attendees
5. **Send Now** or **Schedule**

### Email Integration

Sync with email:
- **Settings** → **Notifications** → **Email**
- Choose what triggers emails:
  - New announcements
  - Event reminders
  - Direct messages
  - Weekly digest

## Troubleshooting

### Cannot Create Club

**Issue**: Create button disabled

**Solutions**:
- Verify you're logged in
- Check if you already have 3 clubs (limit)
- Ensure profile is complete
- Contact admin if issues persist

### Events Not Showing in Calendar

**Issue**: Published events missing

**Solutions**:
1. Refresh page
2. Check date filters
3. Verify event is published (not draft)
4. Clear browser cache

### Members Not Receiving Notifications

**Issue**: Announcements not delivered

**Solutions**:
1. Verify their notification settings
2. Check if they opted out of emails
3. Ensure announcement was sent, not scheduled
4. Ask members to check spam folder

### Permission Denied Error

**Issue**: Can't perform certain actions

**Solutions**:
- Check your role (some features for officers only)
- President may need to grant permissions
- Contact club leadership

## Tips and Best Practices

### Growing Your Club

- **Regular Events** - Consistency builds community
- **Social Media** - Promote on campus platforms
- **Collaborations** - Partner with other clubs
- **Welcome Events** - Monthly orientation for new members

### Keeping Members Engaged

- **Recognition** - Award badges and achievements
- **Variety** - Mix event types and formats
- **Feedback** - Survey members regularly
- **Communication** - Weekly updates

### Event Planning

- **Plan Ahead** - Schedule 2-4 weeks in advance
- **Multiple Options** - Offer different times
- **Backup Plans** - Weather/technical contingencies
- **Follow Up** - Thank attendees, share photos
