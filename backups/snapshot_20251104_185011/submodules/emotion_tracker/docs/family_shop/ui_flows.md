# Family Shop UI Flows & Screens

## 1. Family Shop Screen
- Tab/menu entry for family members
- Shows all item types (avatars, banners, themes, bundles, currency)
- "Pay with Family Wallet" button if allowed
- Handles purchase, approval-needed flows

## 2. Purchase Request Dialogs
- Shows when purchase is pending approval
- Displays status (pending, approved, denied)
- Shows denial reason if applicable

## 3. Admin Dashboard
- Visible to family admins only
- Lists pending purchase requests
- Approve/deny actions
- Real-time updates via WebSocket

## 4. Notifications
- In-app banners/dialogs for real-time events (approval/denial)

## 5. Error Handling
- All error codes mapped to user-friendly messages in dialogs/snackbars
